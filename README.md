# Harbor Backend

## Description:
- A naive harbor backend for basic GET and POST requests to get model data
- Transitioning to Starlette.io soon

## Schema:
Model Schema is as follows:
```
Model
│   README.md
│   file001.txt    
│
└───folder1
│   │   file011.txt
│   │   file012.txt
│   │
│   └───subfolder1
│       │   file111.txt
│       │   file112.txt
│       │   ...
│   
└───folder2
    │   file021.txt
    │   file022.txt
```
