# Harbor Backend

## Description:
- A naive harbor backend for basic GET and POST requests to get model data
- Transitioning to Starlette.io soon

## Schema:
Model Schema is as follows:
```
Model
└─── __id
└─── title
└─── description
└─── params (array)
    └─── key & value object
```
